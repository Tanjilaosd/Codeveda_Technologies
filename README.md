## 📌 Internship Overview

### Levels & Tasks

| Level | Task | Description |
|-------|------|-------------|
| Level 1 (Basic) | Task 1: Setup Development Environment | Install Node.js, npm/yarn, Git, VS Code, and a database. |
| Level 1 (Basic) | Task 2: Build a Simple REST API | Create CRUD API using Node.js & Express. |
| Level 1 (Basic) | Task 3: Frontend with HTML, CSS & JS | Static webpage that interacts with your REST API. |
| Level 2 (Intermediate) | Task 1: Frontend with JS Framework | React/Vue/Angular project with reusable components. |
| Level 2 (Intermediate) | Task 2: Authentication & Authorization | User signup/login using JWT and secure routes. |
| Level 2 (Intermediate) | Task 3: Database Integration | Connect MongoDB/MySQL/PostgreSQL and perform CRUD. |
| Level 3 (Advanced) | Task 1: Full-Stack App (MERN/MEVN/PERN) | Complete web app with frontend-backend integration. |
| Level 3 (Advanced) | Task 2: WebSockets for Real-Time Communication | Implement chat, notifications, or live updates. |
| Level 3 (Advanced) | Task 3: GraphQL API Development | Build GraphQL server with queries, mutations, resolvers. |
